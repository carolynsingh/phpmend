INSTALLING PHP
================

Current doc is for installation on Ubuntu 22.04.

To install PHP :

.. code-block:: bash

   sudo apt-get install php8.1

To check version of php :

.. code-block:: bash

   php -v

.. image:: images/img.png


PHP Syntax

A PHP script starts with <?php and ends with ?>

.. code-block:: php

   <?php
   // PHP code goes here
   ?>
